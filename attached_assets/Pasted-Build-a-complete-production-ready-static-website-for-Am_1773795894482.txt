Build a complete, production-ready static website for "Amani Roots Oils" — a 
premium natural hair care brand founded by a biology and computational 
neuroscience student who blends botanical tradition with modern hair science.

This site must be deployable to Netlify as a static site. Structure it as:
/index.html
/shop.html (placeholder page)
/about.html (placeholder page)
/style.css
/script.js
/assets/ (folder for any local assets)

No frameworks, no build tools, no package.json required. Pure HTML/CSS/JS only.
Netlify will serve index.html as the root. Make sure all links are relative paths.

You have full creative freedom on layout and design decisions. Do NOT make a

"That's exactly the vibe I want — like ASOS, like Zara. Clean and sinful."

📹 HERO SECTION — VIDEO AT THE TOP (MOST IMPORTANT SECTION)
=============================================================

The very first thing visitors see is a full-viewport-height VIDEO HERO.

Use an HTML5 <video> tag with these attributes:
- autoplay
- muted  
- loop
- playsinline
- poster="/assets/hero-poster.jpg" (fallback image placeholder)

Source: <source src="/assets/hero.mp4" type="video/mp4">

The video fills the full screen as a background (object-fit: cover, 
position absolute, z-index 0).

On top of the video, layer:
1. A soft overlay gradient: 
   linear-gradient(to bottom, rgba(45,42,53,0.25) 0%, rgba(45,42,53,0.5) 100%)
   This softens the video without killing it — the motion stays visible.

2. Centered text content (z-index 1, positioned over video):

   Small eyebrow label (all caps, letter-spaced, soft white):
   "ARTISANAL BOTANICAL HAIR CARE"

   Hero headline (massive, Cormorant Garamond, white, elegant):
   "Honoring the Natural Strength of Every Strand"

   Subheadline (light weight, white, max-width 520px centered):
   "Artisanal botanical blends designed to nurture, protect, and celebrate 
   your unique heritage."

   CTA Button:
   "Shop the Collection"
   Style: pill-shaped, background --color-rose (#D4A8C7), white text, 
   no border, soft shadow, hover lifts up 3px + shadow deepens

3. Animate the eyebrow, headline, and subheadline with staggered fade-up:
   - Eyebrow: fadeUp 0.8s ease, delay 0.2s
   - Headline: fadeUp 0.8s ease, delay 0.5s  
   - Subheadline: fadeUp 0.8s ease, delay 0.8s
   - Button: fadeUp 0.8s ease, delay 1.1s

4. Subtle scroll indicator at bottom center: 
   A small animated downward chevron or dot that pulses/bounces gently

NOTE: Since this is a static site, the video file will be added by the client 
later. Build the video element correctly so dropping in hero.mp4 and 
hero-poster.jpg into /assets/ will make it work instantly with no code changes.

⚡ ANIMATIONS MASTER LIST (soft, never jarring)
=============================================================

All animations should feel like petals falling — never like a bounce house.
Easing: ease-in-out or cubic-bezier(0.4, 0, 0.2, 1) everywhere.
Duration: 0.6s–1s for scroll animations. 4s for loops.

1. HERO TEXT: staggered fadeUp (opacity 0→1, translateY 20px→0)
   Eyebrow 0.2s delay → Headline 0.5s → Sub 0.8s → Button 1.1s

2. HERO VIDEO: Ken Burns zoom (scale 1→1.08, 20s ease-in-out infinite)
   Apply to the video element itself for a living, breathing feel

3. NAV: smooth background + blur transition on scroll (0.3s ease)

4. PHILOSOPHY: image translateX(-40px→0) + text fadeUp on scroll entry

5. PILLAR CARDS: staggered scale(0.95→1) + fadeUp, 0.15s apart

6. PRODUCT CARD: gentle float loop translateY(0 → -8px → 0), 4s infinite

7. FAQ ACCORDION: max-height smooth expand + icon rotate(0→45deg)

8. ALL SCROLL ANIMATIONS: use IntersectionObserver with threshold: 0.15
   Add class "visible" to trigger CSS transitions
   Default state: opacity: 0, transform: translateY(20px)
   Visible state: opacity: 1, transform: translateY(0)
   Transition: all 0.7s ease-in-out

9. BUTTON HOVER: translateY(-3px) + box-shadow deepen (0.2s ease)

10. TESTIMONIAL CARDS: on hover, subtle lift + border color shift to 
    --color-accent-1

File structure (flat, Netlify-ready):
/index.html         ← main homepage, all sections
/shop.html          ← placeholder: "Coming Soon" in same style
/about.html         ← placeholder: "Our Story Coming Soon" in same style
/style.css          ← all styles, CSS variables at :root
/script.js          ← nav scroll behavior, IntersectionObserver, FAQ accordion
/assets/            ← empty folder with README.txt:
                       "Drop hero.mp4 and hero-poster.jpg here"

Rules:
- All asset paths must be RELATIVE (./assets/hero.mp4, not /assets/hero.mp4)
- No npm, no node_modules, no build step required
- Netlify drag-and-drop of the entire folder should work instantly
- Include a netlify.toml file at root:

[build]
  publish = "."

[[redirects]]
  from = "/*"
  to = "/index.html"
  status = 200

- All internal links use relative .html paths
- Meta tags in <head>: title, description, og:title, og:description, viewport
- Add <link rel="preconnect"> for Google Fonts for performance
